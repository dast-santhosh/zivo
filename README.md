## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# 1. WHAT IS ZIVO?

## **ZIVO — The AI Companion for Every Student**

**Core definition:**
ZIVO is an AI-powered learning ecosystem designed to help students study smarter, faster, and more efficiently using free, fast AI models.
It combines a personal tutor, notes generator, exam planner, and student community in a single app.

**Reasoning rule:**
*Generalization from existing edTech trends* — products that combine tools + community scale fastest.

---

# 2. ZIVO — CORE PURPOSE

### **Primary Mission:**

Make high-quality academic help accessible to every student in India — without paid tutors or expensive apps.

### **Secondary Mission:**

Automate 80% of student workload using AI: notes, Q&A, planning, revision.

# 3. ZIVO — MAIN FEATURES (Designed for scale)

## **A. ZIVO AI Tutor**

* Answers any question from NCERT / boards
* Step-by-step explanations
* Diagram and chart generation
* Converts tough concepts into simple language

**Reasoning:**
AI tutors reduce dependence on coaching; scalable across millions.

---

## **B. ZIVO Notes Generator**

* Turns any chapter into:

  * flashcards

**Inference rule:**
Task automation → reduced study time → higher retention.

## **C. ZIVO Study Planner**
